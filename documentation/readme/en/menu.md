## This tutorial refers to the English language

# User Menu

The user menu can be accessed by clicking on the username displayed in the top bar.

It contains shortcuts to manage account information and Config Editor preferences.

![Menu](/documentation/images/menu_en.png)

### Editor

Returns to the main configuration file editor.

- Useful when you are on other pages, such as change user, change password, or change language.

![Editor](/documentation/images/editor_en.png)

### Change User

Page to modify the current username.

- Requires entering the new desired username.
- You must provide the current password.
- If 2FA is enabled, the TOTP code will also be requested.
- After saving, the next login must be done with the new username.

![Change user](/documentation/images/change_user.png)

Steps:

1. Enter the new username.
2. Enter the current password.
3. Enter the 2FA code.
  - ⚠️ This field is only requested if two-factor authentication is enabled.
4. Click `Save`.

ℹ️ Note: if you want the name displayed in the authenticator app (e.g., Config Editor: user) to be updated, you must rescan the QR Code available in the Two-Factor Authentication page.

### Change Password

Page to update the login password.

- Requires entering the current password.
- You must type and confirm the new password.
- If 2FA is enabled, the TOTP code will also be requested.
- After saving, the next login must be done with the new password.

![Change Password](/documentation/images/change_password.png)

Steps:

1. Enter the current password.
2. Enter the new password.
3. Confirm the new password.
4. Enter the 2FA code.
  - ⚠️ This field is only requested if two-factor authentication is enabled.
5. Click `Save`.

### Two-Factor Authentication

Page to enable or disable two-factor authentication (TOTP).

- Allows you to activate protection with temporary codes generated by apps like Google Authenticator, Authy, etc.
- Displays a QR Code that must be scanned in the authenticator app.
- Also allows disabling 2FA if it is already enabled.

⚠️ This item only appears if the environment variable **TOTP_ENABLED** is set to `true`.

#### Enable Two-Factor Authentication

![Enable 2FA](/documentation/images/enable_2fa.png)

Steps:

1. Enter the current password.
2. Scan the `QR Code` with your authenticator app.
3. Enter the generated 2FA code.
4. Click `Enable 2FA`.

#### Disable Two-Factor Authentication

![Disable 2FA](/documentation/images/disable_2fa.png)

Steps:

1. Enter the current password.
2. Enter the 2FA code.
3. Click `Disable 2FA`.

### Change Language

Page to select the interface language.

- Currently available in Brazilian Portuguese and English.
- The change is immediate and applied across the entire interface, even if you don’t click Continue.

![Change Language](/documentation/images/change_language.png)

Steps:

1. Select an option from the list.
2. Click `Continue`.
